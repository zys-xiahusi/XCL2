## About

**XCL2** is a Minecraft Java Edition launcher built as a secondary development project based on [PCL2](https://github.com/Hex-Dragon/PCL2). It aims to provide a more powerful, customizable, and community-driven launching experience while respecting the open-source spirit of the original project.

---

## Features (Planned / In Progress)

- [x] Full compatibility with Minecraft Java Edition
- [x] Support for major mod loaders: Forge, Fabric, NeoForge
- [ ] Modern UI with theme customization
- [ ] Advanced profile management and version isolation
- [ ] Built-in mod / resource pack downloader
- [ ] Multi-language support (Simplified Chinese & English)
- [ ] Microsoft / Mojang account integration
- [ ] Dark mode
